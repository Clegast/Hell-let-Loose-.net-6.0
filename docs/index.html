Hell let Loose Artilery

Explanation

We are making a program, that can automatically operate the artillery in the video game "Hell let Loose". The program should recognize a marker, which can be set manually on the map. With the position of the artillery and the marker, the gun will then align itself to fire into the area of the marker.

Information

The program is only working at a resolution from 1920x1080 and when you are not the commander. It is not recomended to play as Officer because the detection of the marker wont find the differece between the markers. Also it expects the game to be on the screen that is set as the primary key in windows. There may be issues if your device is not up to spec.

Manual

Once you are at the artillery you just need to press the "Start" button and tab back in the game. The program is going to open the map and wait for a marker to pop up. Once it found a marker it is going to automatically calculate the best settings and align the artillerys canon. Then the artillery will fire three shots, after that it will start searching markers again. This cycle will be repeated as long as you want.

Settings

artillery position: Here you need to select on which side of the map your artillery is positioned.

artillery type: Select on which team you are.

map north: The difference from the north of the map and the real north in degrees.

runtime: The program will repeat the cicle of searching a marker and shooting for the amounts of minutes you tipe in.

artellery coordinates: pixel coordinates of the artillery in which you currently are.

save settings: if you press this button, the Settings will still be the way you saved them the next time you open the programm.
