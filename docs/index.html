Hell let Loose Artilery

Explanation

We are making a program that can automatically operate the artillery in the video game "Hell let Loose". The program should recognize a marker, which can be set manually on the map. With the position of the artillery gun and the marker, the gun will then align itself to fire into the area of the marker. If we still find time, we will rewrite the program so that it can automatically detect the screen size and adjust the calculations to it. For now, we'll code it with the standard of 1080P.

Information

The Program is only working at a resolution from 1920x1080 and when you are not the Commander. It is not Recomended to play as Officer because the Detection of the marker wont find the differece between the markers. Also it expects the game to be on the screen that is set in windows as the primary screen. There may be issues if your Device is not up to spec.

Manual

Once you are at the artillery you just need to Press the "Start" button and Tab back in the game. The Program is going to Open the map and wait for a Marker to Pop up. Once it found a Marker it is going to automatically Calculate the best settings and align the artillery Canon. after 3 shots its going to end.
