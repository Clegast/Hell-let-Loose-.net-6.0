Hell let Loose Artilery

Explanation

We are making a program that can automatically operate the artillery in the video game "Hell let Loose". The program should recognize a marker, which can be set manually on the map. With the position of the artillery gun and the marker, the gun will then align itself to fire into the area of the marker. If we still find time, we will rewrite the program so that it can automatically detect the screen size and adjust the calculations to it. For now, we'll code it with the standard of 1080P.
